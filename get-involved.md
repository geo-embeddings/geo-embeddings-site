# Get Involved

Geo-Embeddings welcomes contributions from anyone — OGC membership is not required.

## Start Here

The best way to get involved is through GitHub:

1. **Browse open issues** to find topics that interest you
2. **Comment on issues** or open a new one to propose ideas
3. **Submit a pull request** — review the contributing guide below first

## Contributing

- **Keep changes focused** — avoid modifying multiple sections at once; use issues for discussion before large updates
- **Branch in the main repo** for visible collaborative work
- **PRs are reviewed async on GitHub** and merged when rough consensus is reached
